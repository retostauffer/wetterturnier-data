Wetterturnier Wordpress Plugin
===============================

This is the documentation for the 
`wetterturnier.de <http://www.wetterturnier.de>`_
data repository which contains several tools to process and extract
data. The code in this repository is relatively specific but might
be a good starting point if you would like to setup a similar system.
